# Customer Shopping Behavior Analysis
## Project Overview

This project analyzes customer shopping behavior using a dataset of 3,900 transactions to understand purchasing patterns and customer preferences.

The analysis focuses on identifying:

Customer buying behavior

Product category performance

Impact of discounts

Subscription-based purchasing trends

The insights help support data-driven business decisions in retail.

## Dataset

The dataset includes 3,900 customer transaction records with the following fields:

Age

Gender

Product Category

Purchase Amount

Discount Applied

Subscription Status

Payment Method

Shopping Frequency

## Tools Used

Python → Data cleaning and preprocessing

SQL → Data analysis and querying

Power BI → Data visualization and interactive dashboard

## Key Business Questions

Which product categories generate the highest sales?

How do discounts influence customer purchases?

What is the difference between subscription and non-subscription customers?

How do customer demographics affect purchasing behavior?

Which payment methods are most commonly used?

## Dashboard

The Power BI dashboard visualizes key insights, including:

Sales by product category

Customer demographics

Discount impact on purchases

Subscription vs non-subscription analysis

Payment method distribution

Business Insights

Certain product categories contribute the most to total sales.

Discounts increase customer purchasing activity.

Subscription customers show more consistent shopping behavior.

Customer age and gender influence product preferences.
